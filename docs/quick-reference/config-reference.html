Config File

.thailint.yaml # or .thailint.json

Place in project root. Use thailint init-config to create with defaults.

Global Ignore Patterns

ignore: - "__pycache__/" - "*.pyc" - ".git/" - ".venv/" - "node_modules/" - "dist/" - "build/"

Glob patterns applied to all linters. Directories should end with /.

magic-numbers

enabled bool true
allowed_numbers list [-1, 0, 1, 2, 3, 4, 5, 10, 100, 1000]
max_small_integer int 10
Note: max_small_integer allows numbers up to this value in range() and enumerate() calls.

file-placement

file-placement: global_patterns: deny: - pattern: '^(?!src/).*\.py$' message: "Python files go in src/" directories: src: allow: ['.*\.py$'] deny: ['test_.*\.py$'] tests: allow: ['test_.*\.py$']

Uses regex patterns. allow permits matching files, deny blocks them.

Inline Ignore Syntax

LinterIgnore LineIgnore Block
All (except DRY) # thailint: ignore[rule-id] # thailint: ignore-file[rule-id]
DRY only # dry: ignore-next # dry: ignore-block ... # dry: end-ignore
TypeScript/JS // thailint: ignore[rule-id] /* thailint: ignore-file[rule-id] */
Example: # thailint: ignore[srp.violation] on a class definition ignores SRP for that class.
